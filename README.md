# Code With Sagor - Tech Education Landing Page

A professional, animation-rich landing page for the tech education brand "Code With Sagor".

## Features

- Modern dark theme with gradient accents
- Fully responsive design for mobile, tablet, and desktop
- Smooth CSS animations and JavaScript effects
- Typing animation in hero section
- Animated counters for statistics
- Interactive navigation with scroll detection
- Social media integration
- Professional tech startup aesthetic

## How to Use

1. **Replace the logo**: Place your uploaded logo file in the project root directory and name it `logo.png`.

2. **Open the website**: Simply open `index.html` in your web browser.

3. **Customize content**: 
   - Update text content in `index.html`
   - Modify colors in the `:root` section of `style.css`
   - Change typing animation text in `script.js` (typingTexts array)
